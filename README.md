### 一、相关技术

- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`

### 三、部署所需环境

- 后端：`jdk17`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`
- 缓存：`redis`。

### 四、功能描述及功能图展示

系统分为：用户、管理员。

用户功能：1. 首页、2. 文章分类、3. 文章详情、4. 文章评论

管理员功能：1. 登录、2. 管理首页、3. 发布博客、4. 管理博客、5. 分类管理、6. 标签管理、7. 评论管理
